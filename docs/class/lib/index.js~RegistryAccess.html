    RegistryAccess

    Provides registry access to the instance

    This class can either be extended from directly, or mixed in using RegistryAccess.Mixin.

    Example:

    A class that accesses the registry
    import { registry, RegistryAccess } from '@alexlafroscia/service-locator';

registry.register('config', {
  loggedIn: true
});

class SomethingWithRegistryAccess extends RegistryAccess {
  get shouldShowAccount() {
    return this.config.loggedIn;
  }
}

    Test:

    • can define a class that looks something up in the registry
    • it can access nested properties in a registered value
    • it can access properties by building a string
    • it ignores non-string property access

    Allows registry access to be mixed into some other class

    Params:

    Klass T

    the base class to extend

    Return:

    RegistryAccess<T>

    the base class with the registry access mixed in

    Example:

    Extending a base class and gaining registry access
    import { registry, RegistryAccess } from '@alexlafroscia/service-locator';

registry.register('config', {
  loggedIn: true
});

class Foo {
  constructor() {
    this.baseClassProperty = 'foo';
  }
}

class FooWithRegistryAccess extends RegistryAccess.Mixin(Foo) {}

const instance = new FooWithRegistryAccess();
instance.baseClassProperty === 'foo';
instance.config.loggedIn === true;

    Test:

    • can extend a base class and provide registry access
